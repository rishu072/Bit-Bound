# 🔲 Custom QR Code Generator

A simple command-line Python script to generate personalized QR codes from any text or URL.

## 🚀 Quick Start

**Prerequisites:** Install the required library:
```bash
pip install qrcode[pil]
```

**Run the generator:**
```bash
python Qr.py
```

## ✨ Features

- Convert text or URLs into scannable QR codes.
- Customize the foreground and background colors.
- Interactive CLI that saves output instantly as a `.png`.

## 🎮 Usage

Simply run the script and follow the prompts:

```
Enter the test or url for QR code: https://github.com
Enter the file name to save the QR code: Github
Enter the color for the QR code: black
Enter the back color for the QR code: white
```
*Your QR code will be saved in the same directory as `Github.png`.*
